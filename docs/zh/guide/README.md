BF 为前后端分离的单应用后台管理系统。为什么不是微服务呢，因为我不喜欢。（真实原因我不会）。

**BF** 是一个后台管理系统，基于经典技术组合（Spring Boot、Apache Shiro、MyBatis、React），主要目的让开发者注重专注业务，降低技术难度，从而节省人力成本，缩短项目周期，提高软件安全质量。
